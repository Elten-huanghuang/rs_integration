## P3 发现（已知边界行为）

### [P3-1] AutoEatRateLimiter 固定窗口计数

**位置**：`autoeat/AutoEatRateLimiter.java`

**观察**：
自动吃食物的速率限制使用固定时间窗口，与 GuiOpenRateLimiter 类似，可能被边界攻击绕过。

**影响**：
客户端恶意快速触发吃食物可能导致服务器频繁处理，但食物消耗是正确扣除的，不会刷物。

**建议**：
使用滑动窗口或令牌桶算法。

**优先级**：P3（影响有限，仅 DoS 风险）

### [P3-2] FoodEffectBlacklist 效果黑名单计划

**位置**：`autoeat/FoodEffectBlacklist.java`

**观察**：
根据 memory 记录（docs/AUTO_EAT_EFFECT_BLACKLIST_PLAN.md 已删除），曾计划实现效果黑名单功能，但可能已完成或废弃。

**当前状态**：
需确认功能是否已实现，或计划文档是否应恢复。

**优先级**：P3（功能完整性）
